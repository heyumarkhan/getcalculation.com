# PR Creation Template

## Quick Command:
```bash
gh pr create --title "Your PR Title" --body "Your PR description"
```

## With Full Description:
```bash
gh pr create \
  --title "Add Feature Name" \
  --body "## Features
- Feature 1
- Feature 2

## Changes
- Change 1
- Change 2"
```

## Minimal (opens in browser):
```bash
gh pr create --web
```
